# 조직도 생성기 (Organization Chart Generator)

웹 기반 조직도 생성 및 관리 도구입니다.

## 주요 기능

### 관리자 모드 (`index.html`)
- ✏️ 조직도 편집 (노드 추가, 삭제, 이동)
- 🔄 실행 취소/다시 실행
- 📐 자동 배치 및 간격 조정
- 👥 직원 정보 관리
- 💾 버전 관리 (저장/복원)
- 📥 이미지 내보내기 (A3 크기 PNG)
- 💿 데이터 저장/불러오기 (JSON)
- 🔍 확대/축소 (25% - 200%)

### 사용자 모드 (`view.html`)
- 👁️ 조직도 보기 (읽기 전용)
- 🔍 확대/축소
- 📥 이미지 저장

## 사용 방법

### 1. 관리자 모드에서 시작

1. `index.html`을 브라우저에서 엽니다
2. "🔄 초기화" 버튼을 클릭하여 샘플 데이터로 시작하거나
3. 직접 "최상위 부서 추가" 버튼으로 조직도를 만듭니다

### 2. 노드 편집

- **더블클릭**: 노드 정보 편집
- **드래그**: 노드 위치 이동
- **우클릭**: 컨텍스트 메뉴 (하위/형제 부서 추가, 편집, 삭제)

### 3. 연결선 그리기

1. 부모 노드에 마우스를 올립니다
2. 나타나는 연결점(동그라미)을 클릭합니다
3. 자식 노드의 연결점으로 드래그합니다

### 4. 간격 조정

1. "간격 설정" 버튼 클릭
2. 수평/수직 간격, 직원 정보 간격 설정
3. "자동 배치" 버튼으로 적용

### 5. 사용자 모드로 전환

- "👁️ 보기 모드" 버튼을 클릭하면 `view.html`로 이동합니다
- 또는 직접 `view.html`을 엽니다

## 키보드 단축키

### 관리자 모드
- `Ctrl + Z`: 실행 취소
- `Ctrl + Y`: 다시 실행
- `Ctrl + 마우스휠`: 확대/축소
- `Ctrl + +`: 확대
- `Ctrl + -`: 축소
- `Ctrl + 0`: 원래 크기
- `Delete`: 선택된 노드 삭제
- `Esc`: 모달/메뉴 닫기

### 사용자 모드
- `Ctrl + 마우스휠`: 확대/축소
- `Ctrl + +`: 확대
- `Ctrl + -`: 축소
- `Ctrl + 0`: 원래 크기

## 데이터 저장

모든 데이터는 브라우저의 LocalStorage에 자동 저장됩니다.
- 데이터를 백업하려면 "데이터 저장" 버튼으로 JSON 파일로 내보내기
- 데이터를 복원하려면 "데이터 불러오기" 버튼으로 JSON 파일 가져오기
